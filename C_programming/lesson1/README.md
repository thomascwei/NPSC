### 電腦只認得機械碼
- 機械碼就是用0和1組成的程式，但是人類看不懂
- 人類編寫的程式要經過轉譯成機械碼後電腦才能看懂，轉譯有以下三種方式
    * 編譯(compile)
    * 直譯(interpret)
    * 組譯(assembly)
